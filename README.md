# In-Silico Research on Flavonoids in Gandaria with SNEDDS

## Overview

This repository contains comprehensive research on **OPSI (In-Silico)** studies investigating the anti-diabetes properties of **Gandaria seed extract** utilizing a **SNEDDS (Self-Nanoemulsifying Drug Delivery System)** formulation. The project combines computational molecular docking studies with pharmaceutical research to evaluate the potential of Gandaria flavonoids as anti-diabetic agents.

## Project Objectives

- **Characterization**: Identify and analyze flavonoid compounds present in Gandaria seed extract
- **Molecular Docking**: Perform computational studies on key anti-diabetes enzymes (Alpha-amylase)
- **SNEDDS Development**: Design an optimized self-nanoemulsifying drug delivery system for improved bioavailability
- **Drug Discovery**: Evaluate the potential of Gandaria-derived compounds as novel anti-diabetic therapeutics
